Was war in Python ähnlich wie in Bash?
Python war in manchen Bereichen ähnlich wie Bash, zum Beispiel bei Variablen, Eingaben und Ausgaben. Trotzdem finde ich, dass die beiden Sprachen nicht sehr ähnlich sind.

Was war in Python einfacher zu lesen?
Die Syntax in Python ist einfacher zu lesen. Vielleicht liegt das auch daran, dass ich bereits daran gewöhnt bin.

Was war am Anfang ungewohnt?
Für mich war am Anfang nichts besonders ungewohnt.

Warum ist Einrückung in Python wichtig?
Die Einrückung ist wichtig, weil sie Python zeigt, welche Anweisungen zu einem Block gehören, zum Beispiel bei if-Abfragen oder Schleifen.

Welche Aufgabe aus dem IT-Support könnte man später mit Python automatisieren?
Man könnte die Ticketaufnahme und die Dokumentation von Problemen von Kunden oder Mitarbeitern automatisieren. Dadurch könnten Supportanfragen schneller erfasst und verarbeitet werden.
